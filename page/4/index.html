CF161D - Distance in Tree

发表于 2024-02-26 更新于 2024-03-14 分类于 ACM ， Codeforces ， 161

Problem Description

一棵树是一个不包含任何环的连通图。

树的两个顶点之间的距离是这些顶点之间最短路径的长度（以边为单位）。

给定一棵有 $n$ 个顶点的树和一个正数 $k$ 。找到具有距离正好为 $k$ 的顶点对的不同对数。请注意，对 $(v, u)$ 和 $(u, v)$ 被视为相同的一对。

输入

  • $n\le 50000, k\le 500$
  • 接下来 $n-1$ 行，每行两个正整数 $a_i,b_i$，表示 $a_i,b_i$ 之间有一条边。

输出

输出一个整数，表示树上有多少点对，满足他们的距离为 $k$。

Solution

这是一道点分治的模板题。点分治适合处理大规模树上路径问题。

我们先选定树上的一个节点 $u$ 作为根，那么我们可以吧树上的所有路径划分成两类：

  1. 经过 $u$ 的路径
  2. 不经过 $u$ 的路径

而由于父亲-儿子的关系，我们可以使用递归处理第二类路径。我们所要做的，就是考虑如何处理第一类路径。

考虑两个点 $u,v$，他们之间的路径经过根节点 $rt$，那么我们可以将这条路径拆成两部分看：一部分是 $u\to rt$，另一部分是 $rt\to v$。如果 $u\to v$ 是一条合法的路径，那么也就是说 $d_{u\to rt}+d_{v\to rt}=K$。

如果给定一个根节点，我们可以通过 $\text{DFS}$ 预先处理好 $dis$ 数组（$dis[i]$ 表示到根节点距离为 $i$ 的节点个数），这样就方便了后续的计算。假定当前树的根节点是 $x$，那么我们枚举 $i\in [0,K]$，$i$ 表示路径的一端到 $x$ 距离为 $i$，那么另一端的距离就是 $K-i$，把 $dis[i]$ 和 $dis[K-i]$ 相乘即可。

代码中 dis[d[u]]++ 是统计节点数的，统计距离根节点为 $i$ 的点数有几个。并且加入了一个剪枝优化：如果当前的 d[u] 已经超过 $K$，那么 $u$ 的子节点到根节点的距离也一定大于 $K$，不会对答案产生贡献。

计算以 $u$ 为根节点时有多少方案数。要注意的是，如果 $K$ 是偶数，那么就会在 $dis[\frac{K}{2}]$ 中选两个点，方案数并不是直接自己乘自己。

这样存在一个小bug，如果同时被算入 $dis[i]$ 和 $dis[K-i]$ 的两个点在同一棵子树上，这样的路径其实是不合法的，因为路径没有经过根节点。所有要减去这一部分的方案。但是我们发现他们到根节点的路径都经过根节点的某个子节点，这个子节点到根节点的距离都是 $1$。所以我们可以用 calc(v, 1, -1) 算出不经过根节点的方案数减一下即可。

但是这样会有一个问题：如果树的形态是一条链，那么如果根节点选到了链的一端，每次递归下去就相当于只递归到链上的下一个点，整体的时间复杂度会达到 $O(N)$。所以，我们在预处理之前，需要先求出树的重心，然后将树的中心作为根节点进行递归。

注意此处的 if 判断中是 sum/2 （sum表示子树的大小），是因为我们要找的是子树的根节点，而不是整棵树的根节点。

ABC340G - Leaf Colors

发表于 2024-02-25 更新于 2024-03-14 分类于 ACM ， AtCoder Beginner Round ， 340

Problem Description

ABC340G - Leaf Colors

题面翻译

给出一棵节点数为 $N$ 的树 $T=(V,E)$，节点有颜色，节点 $i$ 的颜色为 $A_i$。

求满足以下条件的 $V$ 的子集 $S$ 的数量：

  • $S$ 的导出子图 $G$ 是一棵树；
  • $G$ 的叶节点颜色一致。

答案对 $998244353$ 取模。

导出子图：点集 $S$ 的导出子图 $G$ 的点集为 $S$，边集为 ${(u,v)|u,v\in S\land (u,v)\in E}$。

输入格式

入力は以下の形式で標準入力から与えられる。

$N$
$A_1$ $A_2$ $\dots$ $A_N$
$u_1$ $v_1$
$u_2$ $v_2$
$\vdots$
$u_{N-1}$ $v_{N-1}$

输出格式

問題文の条件を満たす頂点集合の(非空な)部分集合 $S$ の個数を $998244353$ で割った余りを出力せよ。

制約

  • $1\ \leq\ N\ \leq\ 2\ \times\ 10^5$
  • $1\ \leq\ A_i\ \leq\ N$
  • $1\ \leq\ u_i\ \lt\ v_i\ \leq\ N$
  • 入力で与えられるグラフは木
  • 入力される値は全て整数

Solution

我们首先要考虑到计数DP。先简化问题，由于对于一棵树而言，只有根节点、叶子节点才有可能度数为 $1$，如果让叶子节点的颜色固定，那么个数是比较好求的。

我们的大致思路是：枚举颜色 $C$，然后求叶子节点颜色都是 $C$ 的树的个数。

然后我们可以发现，假定枚举的颜色是 $C$，如果某节点 $V_0$ 的颜色不是 $C$，那么 $V_0$ 不可能作为叶子节点，所以，我们可以建立虚树，只在颜色为 $C$ 的节点上建树。为了保证虚树是连通的，我们添加他们之间的 $LCA$.

然后我们枚举颜色 $C$。考虑 $dp[u]$ 表示：选择了 $u$ 时，有多少树是以 $u$ 为根、且所有叶子节点的颜色都是 $C$. 所以， $dp[u]=\prod_{v\in son[u]}(dp[v]+1)$. 意义是，只要叶子节点相同，可以看成是子树 $v_1$ 与子树 $v_2$ 通过 $u$ 连在一起，形成一张新的图，并且这张图是一棵树； $+1$ 是因为还可以不选择子树 $v$ 内的任何节点，单独一个 $u$ 也是合法的答案。

但是，还要考虑到 $u$ 的颜色不是 $C$ 的情况。如果 $u$ 的颜色不是 $C$，那么如果只选择一棵 $u$ 的子树和 $u$ 组成新的图，在这张新的图中， $u$ 的度数也是 $1$，但 $u$ 的颜色不是 $C$，这个答案是不合法的；此外，如果都不选，单独 $u$ 本身也不能成为合法的解。所以，总共有 $(\sum dp[v])+1$ 种不合法的解。

ABC340F - S=1

发表于 2024-02-25 更新于 2024-03-14 分类于 ACM ， AtCoder Beginner Round ， 340

Problem Description

ABC340F - S=1

题面翻译

给定整数 $X,Y$，求一个整数二元组 $(A,B)$，满足以下条件：

  • $|A|,|B|\le 10^{18}$
  • $(0,0),(X,Y),(A,B)$ 三点所构成的三角形面积为 $1$。

若不存在满足以上条件的二元组 $(A,B)$，输出 -1，否则输出一行两个整数 $A,B$。

$|X|,|Y|\le 10^{17},(X,Y)\not= (0,0)$

输入格式

入力は以下の形式で標準入力から与えられる。

$X\ \ Y$

输出格式

条件を満たす整数の組 $(A,\ B)$ が存在する場合は以下の形式で出力せよ。

$A\ \ B$

条件を満たす整数の組 $(A,\ B)$ が存在しない場合は -1 を出力せよ。

制約

  • $-10^{17}\le X,Y\le 10^{17}$
  • $(X,\ Y)\ \ne\ (0,\ 0)$
  • $X,\ Y$ は整数

Solution

我们先使用行列式计算三角形的面积。

$$
\frac{1}{2}\begin{vmatrix}0&0&1\X&Y&1\A&B&1\end{vmatrix}=1
$$

我们将其化简得到：$BX-AY=\pm 2$。观察到这很类似于扩展欧几里德 $ax+by=\gcd(a,b)$。

因为 $A,B$ 都是正整数，如果 $\gcd(X,Y)\gt 2$，那么就是无解的。否则，我们可以先找到一组 $(A_1,B_1)$ 满足 $BX-AY=\gcd(X,Y)$，然后再两边同乘 $\frac{2}{\gcd(X,Y)}$ 就可以得到一组合法的答案了。

ABC335E - 不递增路径

发表于 2024-02-25 更新于 2024-03-14 分类于 ACM ， AtCoder Beginner Round ， 335

Problem Description

ABC335E - 不递增路径

問題文

$N$ 頂点 $M$ 辺の連結な無向グラフがあり、 $i$ 番目の辺は頂点 $U_i$ と頂点 $V_i$ を双方向に結びます。

また、全ての頂点に整数が書いてあり、頂点 $v$ には整数 $A_v$ が書かれています。

頂点 $1$ から頂点 $N$ への単純なパス ( 同じ頂点を複数回通らないパス ) に対して、以下のように得点を定めます。

  • パス上の頂点に書かれた整数を通った順に並べた数列 を $S$ とする。
  • $S$ が広義単調増加になっていない場合、そのパスの得点は $0$ である。
  • そうでない場合、 $S$ に含まれる整数の種類数が得点となる。

頂点 $1$ から頂点 $N$ への全ての単純なパスのうち、最も得点が高いものを求めてその得点を出力してください。

  • $S$ が広義単調増加であるとは?

    長さ $l$ の数列 $S=(S_1,S_2,\dots,S_l)$ が広義単調増加であるとは、 全ての整数 $1 \le i < l$ について $S_i \le S_{i+1}$ を満たすことを言います。

制約

  • 入力は全て整数
  • $2 \le N \le 2 \times 10^5$
  • $N-1 \le M \le 2 \times 10^5$
  • $1 \le A_i \le 2 \times 10^5$
  • グラフは連結である
  • $1 \le U_i < V_i \le N$
  • $i \neq j$ なら $(U_i,V_i) \neq (U_j,V_j)$

Solution

因为此题中，一条递增路径上点权大小相同的点不会对答案贡献多次，而且，由于递增且相同，所以这些点一定上路径是连续的，也就是说有边相连。那么，我们可以考虑用并查集将一条边连着的、点权相同的点合并起来。

此题中，要求递增路径，我们可以想到将无向边拆开看成 $u→v,v→u$ 的两条有向边，然后只保留点权更小的点连向点权更大的点的有向边（如果 $A_u<A_v$，那么保留 $e(u→v)$）

【注意！】 这里按点的大小进行松弛，让 $ans[i]$ 表示到 $i$ 的所有递增路径中最多有多少不同点权。如果到达 $i$ 有递增路径，那么 $ans[i]>0$，否则 $ans[i]<0$。如果到 $i$ 已经没有递增路径，那么从 $1 \rightarrow i \rightarrow n-1$ 也一定不会有递增路径（ $1 \rightarrow i$ 已经不递增了），这是 $\text{ans}$ 的意思。

  • 然后考虑点权个数。对于将要松弛的边 $e(u,v)$，起点点权 $v[u]$：如果到 $u$ 有递增路径（$ans[u]>0$），那么将 $u$ 添加到这个路径上，可以使答案 $+1$
  • 否则若 $ans[u]<0$，两种情况：到 $u$ 没有递增路径，或者还没更新过 $u$。第一种情况，如果没有递增路径，即使把 $u$ 加上去也不会有任何效果，答案还是 $0$。如果还没更新过 $u$，那么就一定要用起点点权更大的边 $e_1$ 去更新 $u$，这也会导致 $u$ 没有递增路径。
  • 所以只需要考虑 $ans[u]>0$ 的时候更新即可。
