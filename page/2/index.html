Vanilla Self Attention

Self-Attention 就是将输入的 Token 再和自己做一次加权平均，也就是说字典 D={(k=xi,v=xi)}i=1n\mathcal{D}=\{(\mathbf k=\mathbf x_i,\mathbf v=\mathbf x_i)\}_{i=1}^nD={(k=xi​,v=xi​)}i=1n​，Align 完的输出可以记作 yi\mathbf y_iyi​，则有

yi=f(xi,D)∈Rd\mathbf y_i=f\Big(\mathbf x_i,\mathcal{D}\Big)\in \mathbb R^d yi​=f(xi​,D)∈Rd

也就是说，给定一个输入 Token xix_ixi​，先经过 Embedding 转为 aia_iai​ 词向量，再分别乘上 Wq,Wk,WvW^q,W^k,W^vWq,Wk,Wv 就将输入的 Token 映射到可以进行 Attention 计算的 Q,K,VQ,K,VQ,K,V 向量了，将多个这样的向量拼起来就是 Q,K,VQ,K,VQ,K,V 矩阵了。

计算完 QK⊺dV\frac{QK^\intercal}{\sqrt d}Vd​QK⊺​V，我们按行进行 softmax⁡\operatorname{softmax}softmax

Multi Head Self Attention

假设分为 hhh 个 Attention Head，我们其实要做的就是把 qqq 均分成 hhh 个部分，分别交给 hhh 个 Self Attention 进行计算。

问题

多头注意力机制 Multi-Head Attention

Posted on 2025-01-27 Edited on 2025-01-28 In 深度学习、机器学习 , 大语言模型 , 注意力机制

有的时候，给定同样的 Q, K, V 的输入，我们希望模型能够学习到不同的知识，或者能够结合不同的知识。

所以，我们可以让这些相同的 Q, K, V 经过不同的 projection 之后输入到 hhh 个不同的 Attention 里面，平行地进行 Attention Pooling，最后再收集这些输出，拼接在一起再经过一个线性变换，得到最终的输出。这个机制被称作 Multi-Head Attention
