【图论】2-sat

2-SAT

SAT 是一类问题，给定一个包含 nnn 个变量的布尔表达式，你需要给这 nnn 个变量分别赋值 true false 使得表达式的结果为 true。一个布尔表达式是多个子句 (clause)的 and，每个子句又是若干个变量的 or。

而 2-SAT 是一类特殊的 SAT 问题，每一个子句里有且恰有 222 个变量。这个限制条件使得 SAT 这一类 NP 完全问题可以在 O(n+m)O(n+m)O(n+m) 的时间内求解。

举一个布尔表达式的例子：

(a∨¬b)∧(¬a∨b)∧(¬a∨¬b)∧(a∨¬c)(a \lor \lnot b) \land (\lnot a \lor b) \land (\lnot a \lor \lnot b) \land (a \lor \lnot c) (a∨¬b)∧(¬a∨b)∧(¬a∨¬b)∧(a∨¬c)

求解

判断有解

  1. 如果从 xxx 出发可以到达 ¬x\lnot x¬x 且 ¬x\lnot x¬x 可以到达 xxx，那么此时是无解的，反之亦然（其实是充要条件）。我们假定 x  ⟹  y1  ⟹  y2  ⟹  ⋯  ⟹  ¬xx\implies y_1\implies y_2\implies \dots\implies \lnot xx⟹y1​⟹y2​⟹⋯⟹¬x，如果 x=x=x= true，那么因为只有 true   ⟹  \implies⟹ true === true，所以，y1=y2=⋯=¬x=y_1=y_2=\dots=\lnot x=y1​=y2​=⋯=¬x= true，显然矛盾。

这里，“能够互相到达”这个条件可以联想到“强连通分量”这一知识点。没错，我们可以应用强连通分量的 Tarjan SCC 算法在 O(n+m)O(n+m)O(n+m) 的时间内验证 xxx 与 ¬x\lnot x¬x 是否联通。

可行解

当然，有可能存在 xxx 可以到达 ¬x\lnot x¬x 或者 ¬x\lnot x¬x 可以到达 xxx，此时 xxx 不能随意取值。

取值策略

令 comp[v]comp[v]comp[v] 表示原图里点 vvv 所在的 SCC 编号

  1. 把所有 SCC 拓扑排序
  2. 如果 comp[x]<comp[¬x]comp[x]\lt comp[\lnot x]comp[x]<comp[¬x] 那么 x=x=x= false；否则 x=x=x= true

常见转化

  1. xxx 必须等于 true/false
    • 等价于 x∨xx\lor xx∨x / ¬x∨¬x\lnot x \lor \lnot x¬x∨¬x
  2. x,yx,yx,y 中有且仅有一个为 true
    • 等价于 (x∨y)∧(¬x∨¬y)(x\lor y)\land(\lnot x\lor\lnot y)(x∨y)∧(¬x∨¬y)
  3. x,yx,yx,y 中至少一个为 true
    • 等价于 x∨yx\lor yx∨y
  4. x,yx,yx,y 必须有相同的值
    • 等价于 (¬x∨y)∧(x∨¬y)(\lnot x\lor y)\land(x\lor\lnot y)(¬x∨y)∧(x∨¬y)

例题

CF1971H

要求替换后的矩阵按列从小到大排序后，最中间一行全为 +1+1+1。也就是说，每一列至少要有两个 +1+1+1。

同时可以感觉到（？）+1,−1+1,-1+1,−1 的两个状态比较类似 2-SAT，我们的确可以用 2-SAT 解决。把 −1-1−1 看成 false，111 看成 true，三个数 (x,y,z)(x,y,z)(x,y,z) 至少有两个 111 等价于：x,yx,yx,y 中至少一个 111 且 y,zy,zy,z 中至少一个 111 且 x,zx,zx,z 中至少一个 111。这样，我们就可以套用 2-SAT 来求解了。

首先对于一个旗子 iii，要么选择插在 xix_ixi​，要么插在 yiy_iyi​，这里可以看到 2-SAT 的影子。

我们考虑如何找出 max⁡d\max dmaxd。我们可以二分答案，然后转化为判定问题。2-SAT 判定是 O(n)O(n)O(n) 的，二分是 O(log⁡x)O(\log x)O(logx) 的，但是2-SAT 的建图如果按朴素方法建图是 O(n2)O(n^2)O(n2) 的。

我们观察到对于两个旗子 i,ji,ji,j 和枚举的答案 DDD

  1. 如果 ∣xi−xj∣<D|x_i-x_j|<D∣xi​−xj​∣<D，说明 iii 插在 xix_ixi​ 的话，jjj 就不能插在 xjx_jxj​，必须插在 yjy_jyj​，反之亦然。因此 2-SAT 里可以看成是 xi  ⟹  yjx_i\implies y_jxi​⟹yj​
  2. 如果 ∣xi−yj∣<D|x_i-y_j|<D∣xi​−yj​∣<D，说明 iii 插在 xix_ixi​ 的话，jjj 就不能插在 yjy_jyj​，必须插在 xjx_jxj​，反之亦然。xi  ⟹  xjx_i\implies x_jxi​⟹xj​
  3. 如果 ∣yi−xj∣<D|y_i-x_j|<D∣yi​−xj​∣<D，说明 iii 插在 yiy_iyi​ 的话，jjj 就不能插在 xjx_jxj​，必须插在 yjy_jyj​，反之亦然。yi  ⟹  yjy_i\implies y_jyi​⟹yj​
  4. 如果 ∣yi−yj∣<D|y_i-y_j|<D∣yi​−yj​∣<D，说明 iii 插在 yiy_iyi​ 的话，jjj 就不能插在 yjy_jyj​，必须插在 xjx_jxj​，反之亦然。yi  ⟹  xjy_i\implies x_jyi​⟹xj​
