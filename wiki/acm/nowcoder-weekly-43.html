这一场比较简单，两道语法，两道基础算法，两道计算几何。

A. 小红平分糖果

签到题。是偶数就 ok，否则 −1-1−1.

B. 小红的完全平方数

先找到离 xxx 最近的两个完全平方数，保证奇偶性相同。然后考虑到先 +2+2+2 再 −2-2−2 相当于不变，因此要么一直 +2+2+2 要么一直 −2-2−2。分别计算距离哪个平方数更近即可。

计算 <x<x<x 的奇偶性相同的平方数：ll a = sqrt(n); if ((a & 1) != (n & 1)) a--;

C. 小苯的字符串变化

由于要求前一半全大写，后一半全小写，我们可以枚举大写小写的分界点，用前缀和判断需要修改几个字符。

D. 小红的子数组排列判断

因为数组是连续的 kkk 个，这和滑动窗口的思想比较类似。所以我们维护一个滑动窗口，用 std::set 维护窗口内没有出现的元素。那么，如果 set 为空，说明刚好是 1∼k1\sim k1∼k 的排列，每次拓展窗口，cnt 计数器 +1+1+1 把这个数从 set 中删掉；缩小窗口时，cnt 计数器 −1-1−1，如果减到 000，那么重新放入 set。

时间复杂度：每个元素插入、弹出 set 至多一次，时间复杂度 O(nlog⁡n)O(n\log n)O(nlogn)。

E. 小红的平行四边形

【前置数学知识】

平行四边形由向量 v1=(a,b)v_1=(a,b)v1​=(a,b) 和 v2=(c,d)v_2=(c,d)v2​=(c,d) 构成，那么平行四边形的面积为

A=∣v1×v2∣=∣ad−bc∣A=|v_1\times v_2|=|ad-bc| A=∣v1​×v2​∣=∣ad−bc∣

首先可以想到一个朴素的 O(n3)O(n^3)O(n3) 算法。先 O(n2)O(n^2)O(n2) 枚举两个点，算出一个向量 v1v_1v1​，然后再 O(n)O(n)O(n) 枚举第三个点，判断第四个点是否存在于给定的点里，在的话，就直接用平行四边形面积公式计算。

不过这样会超时。如果一个向量边已知，那么能够组成平行四边形的对边向量也一定是这个。既然一条边是已知的，那么面积就取决于对边之间的距离——也可以说两条直线之间的距离，我们就希望这个距离尽可能远。

那么怎么做呢？既然向量已知，我们可以用一条基准直线，计算其他直线到这条基准直线的有符号距离（+++ 表示在右侧，−-− 表示在左侧，且绝对值越大，离这条直线越远），按有符号距离排序，然后取出开头末尾的两个元素，就是以该向量为一条边的平行四边形中，面积的最大值。

时间复杂度分析：n=1000n=1000n=1000，最多有 O(n2)O(n^2)O(n2) 个向量，最坏情况下全部平行，排序需要 O(n2log⁡n2)=O(n2log⁡n)O(n^2\log n^2)= O(n^2\log n)O(n2logn2)=O(n2logn)。因此总的时间复杂度是 O(n2log⁡n)O(n^2\log n)O(n2logn)

F. 小红小紫画线

容斥原理可解。具体详见 评论区题解

记 𝑆=∑iai𝑆=\sum_ia_iS=∑i​ai​，甲选的两点为 A,BA,BA,B，乙选的两点为 C,DC,DC,D（A、B不分辨，C、D不分辨）。

  1. 先考虑两条线段端点不重合的情况：

    先不考虑多边形的限制，只需满足 ABABAB 与 CDCDCD 有交叉点，这一选法相当于在多边形上取 444 个点，再将 444 个点按 ACBDACBDACBD 或 CADBCADBCADB 的顺序排列，总方案数为

    N1=2×(S4)N_1=2\times\binom{S}{4} N1​=2×(4S​)

    1. 从中需要排除掉 ABABAB 或 CDCDCD 其一在同一条边上的情况。这一选法相当于先取一条边 aia_iai​，在 aia_iai​ 上取 333 个点，将 333 个点按 ACBACBACB 或 CADCADCAD 的顺序排列，再在 aia_iai​ 以外的边上取 111 个点，方案数为

      N2=2×∑i(S−ai)(ai3)N_2=2\times\sum_i(S-a_i)\binom{a_i}{3} N2​=2×i∑​(S−ai​)(3ai​​)

    2. 还需要排除 ABABAB 与 CDCDCD 均在同一条边上的情况。这一选法相当于先取一条边 aia_iai​，在 aia_iai​ 上取 444 个点，再将 444 个点按 ACBDACBDACBD 或 CABDCABDCABD 的顺序排列，方案数为

      N3=2×∑i(ai4)N_3=2\times\sum_i\binom{a_i}{4} N3​=2×i∑​(4ai​​)

  2. 再考虑两条线段有一个端点重合的情况：

    先固定两个重合的端点，考虑另外两个端点，这一选法相当于先取一条边 aia_iai​，在 aia_iai​ 上取 111 个点为 A=CA=CA=C，再在 aia_iai​ 以外的边上取 222 个不同点为 B,DB,DB,D，方案数为

    N4=2×∑iai(S−ai2)N_4=2\times\sum_ia_i\binom{S-a_i}{2} N4​=2×i∑​ai​(2S−ai​​)
