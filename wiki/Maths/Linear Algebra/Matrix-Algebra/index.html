Determinant of a Matrix

Theorem: Determinant for Matrix Operation

Suppose AAA is a square matrix. Let BBB be the matrix obtained from AAA by swapping 2 rows or columns. Then

det⁡B=−det⁡A\det B=-\det AdetB=−detA

Let CCC be the square matrix obtain from AAA by multiplying a single row or column by scalar rrr, then

det⁡C=rdet⁡A\det C=r\det AdetC=rdetA

If AAA has 2 same rows or 2 same columns, then

det⁡A=0\det A=0detA=0

Let DDD be the matrix obtained from AAA by multiplying a row/column by scalar rrr and then adding it to another row/column, then

det⁡D=det⁡A\det D=\det AdetD=detA
