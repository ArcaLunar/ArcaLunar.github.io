Vision Transformer

Posted on 2025-01-29 Edited on 2025-01-30 In 深度学习、机器学习 , 计算机视觉 , ViT
随着 Transformer 在 LLM 领域的大放异彩，计算机视觉也引入了这一模型，称为 Vision Transformer (ViT)

考虑一张图像 x∈Rh×w×c\mathbf x\in\mathbb R^{h\times w\times c}x∈Rh×w×c，令 patch 的图像大小为 p×pp\times pp×p，那么图像就被分割为 m=hwp2m=\frac{hw}{p^2}m=p2hw​ 个 patch，每一个 patch 都被展平为一个长度为 cp2cp^2cp2 的向量。于是乎每个向量都可以看作是一个 Token 输入 Transformer Encoder.

我们向 Patch Token Sequence 前额外加一个特殊字符 <cls>，和这 mmm 个 patch 分别经过 projection，形成 m+1m+1m+1 个向量的 Sequence，并且进行位置编码。
