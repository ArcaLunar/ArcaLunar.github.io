数学分析一 Ep.1 ：实数的公理化描述、四条公理

实数

  • 我们定义实数 R\RR 是一个集合，上面有两个操作：
    • 加法 +:R×R↦R,(x,y)↦x+y+:\R\times\R\mapsto\R, (x,y)\mapsto x+y+:R×R↦R,(x,y)↦x+y
    • 乘法 ⋅:R×R↦R,(x,y)↦x⋅y\cdot: \R\times\R\mapsto\R, (x,y)\mapsto x\cdot y⋅:R×R↦R,(x,y)↦x⋅y
  • 同时还有序关系 (Order Relation) ≤:x≤y\le: x\le y≤:x≤y.

域公理 (Axioms of Field)

  1. 加法交换律：x+y=y+xx+y=y+xx+y=y+x
  2. 加法结合律：x+(y+z)=(x+y)+zx+(y+z)=(x+y)+zx+(y+z)=(x+y)+z
  3. 加法单位元：存在 0∈R0\in\R0∈R 使得对 ∀x∈R\forall x\in\R∀x∈R 有 0+x=x0+x=x0+x=x 成立
  4. 加法逆元的存在性、唯一性：对 ∀x∈R\forall x\in\R∀x∈R，存在且只存在一个 −x∈R-x\in\R−x∈R，使得 x+(−x)=0x+(-x)=0x+(−x)=0

    这里，我们还没有证明 (−1)⋅x=−x(-1)\cdot x=-x(−1)⋅x=−x，−x-x−x 整体应该被当作一个记号，用于表示 xxx 的加法逆元。

  5. 乘法结合律：x⋅(y⋅z)=(x⋅y)⋅zx\cdot (y\cdot z)=(x\cdot y)\cdot zx⋅(y⋅z)=(x⋅y)⋅z
  6. 乘法交换律：x⋅y=y⋅xx\cdot y=y\cdot xx⋅y=y⋅x
  7. 乘法单位元：存在 1∈R,1≠01\in\R,1\ne 01∈R,1=0，使得对任意 ∀x∈R\forall x\in\R∀x∈R 都有 1⋅x=x1\cdot x=x1⋅x=x
  8. 乘法逆元的存在性、唯一性：对任意 ∀x∈R−{ 0 }\forall x\in\R-\set{0}∀x∈R−{0}，存在 x−1∈Rx^{-1}\in\Rx−1∈R 使得 x⋅x−1=1x\cdot x^{-1}=1x⋅x−1=1
  9. 乘法分配律：x⋅(y+z)=x⋅y+x⋅zx\cdot (y+z)=x\cdot y+x\cdot zx⋅(y+z)=x⋅y+x⋅z

序公理 (Axioms of Order)

  1. 序的传递性：x≤y,  y≤z  ⟹  x≤z.x \le y,\; y \le z\;\Longrightarrow\;x \le z.x≤y,y≤z⟹x≤z.
  2. 序可以决定元素：x≤y,  y≤x  ⟹  x=yx\le y,\; y\le x \implies x=yx≤y,y≤x⟹x=y
  3. 全序关系：∀x,y∈R,  x≤y\forall x,y\in\R,\; x\le y∀x,y∈R,x≤y 或者 y≤xy\le xy≤x 二者至少有一个正确
  4. 与加法相容：x≤y  ⟹  x+z≤y+zx\le y\implies x+z\le y+zx≤y⟹x+z≤y+z
  5. 与乘法相容：x≥0∧y≥0  ⟹  xy≥0x\ge 0\land y\ge 0 \implies xy\ge 0x≥0∧y≥0⟹xy≥0

证明 反证法：若 1<01\lt 01<0，则 −1>0-1\gt 0−1>0，由于和乘法相容，有

1⋅(−1)<0⋅(−1)−1<0 1\cdot (-1)\lt 0\cdot (-1)\\ -1\lt 0 1⋅(−1)<0⋅(−1)−1<0
