D - Minimize Range

Suppose we always sort they array a[]a[]a[] increasingly. Then min⁡a=a1,max⁡a=an\min a=a_1, \max a=a_{n}mina=a1​,maxa=an​

First of all, we can always conclude that max⁡−min⁡<k\max-\min < kmax−min<k, because otherwise, we can add kkk to a1a_1a1​ to decrease the difference. This gives a1+k>ana_1+k>a_na1​+k>an​.

Now, the minimum possible answer can only be attained by increasing a1a_1a1​, because if we add kkk to a2a_2a2​, then either the difference remains the same, or a2+k>ana_2+k>a_na2​+k>an​ and the difference cannot be smaller. Given that a1+k>ana_1+k>a_na1​+k>an​, it’ll take at most O(n)O(n)O(n) times of +k+k+k operation to increase all elements by kkk.
