CF896 - [C] 珂朵莉树模板题

发表于 2024-03-31 分类于 ACM ， Codeforces ， 896

题目

题解

珂朵莉树的核心思想是把值相同的区间合并成一个节点放在 std::set 里。set 里每一个节点的保存方式是：

[l,r][l,r][l,r] 表示这个节点管辖的区间，v 是自己指定的附加数据（如区间的颜色信息）。

split 操作

split 是最核心的操作之一，它用于将原本包含点 xxx 的区间（设为 [l,r][l, r][l,r]）分裂为两个区间 [l,x−1][l, x-1][l,x−1] 和 [x,r][x, r][x,r] 并返回指向后者的迭代器。

注：珂朵莉树在进行求取区间左右端点操作时，必须先 split 右端点，再 split 左端点。若先 split 左端点，返回的迭代器可能在 split 右端点的时候失效，可能会导致 RE。

assign 操作

assign 操作用于对一段区间进行赋值，使珂朵莉树的 set 大小减小，保证时间复杂度。

其他操作

其实就是套模板，对拆分出来的区间进行操作。
