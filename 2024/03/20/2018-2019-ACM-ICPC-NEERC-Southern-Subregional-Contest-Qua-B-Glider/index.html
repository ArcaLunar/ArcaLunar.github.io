思路一：贪心 + 滑动窗口

飞机飞过的区间一定是连续的，而且贪心地想，如果飞机能跨过下坠区间，那么就能在动力区间继续飞行更远的距离，所以，我们可以使用滑动窗口贪心地加入区间，只要还能继续飞行，就继续飞行；否则，弹出最前面的下坠区间（和对应的动力区间），直到能够跨过这段下坠区间继续飞行。然后加入滑窗。

最重要的是发现可以连续贪心的性质

AC Code

思路二：二分

我们肯定是从某个动力区间的左端点出发（这样才不浪费下坠距离），所以我们用前缀和处理从区间 aaa 飞到区间 bbb 要下坠多少距离，二分找到最后一个不会下坠坠地的区间右端点，计算答案。
