ABC340F - S=1

发表于 2024-02-25 更新于 2024-03-14 分类于 ACM ， AtCoder Beginner Round ， 340

Problem Description

ABC340F - S=1

题面翻译

给定整数 $X,Y$，求一个整数二元组 $(A,B)$，满足以下条件：

  • $|A|,|B|\le 10^{18}$
  • $(0,0),(X,Y),(A,B)$ 三点所构成的三角形面积为 $1$。

若不存在满足以上条件的二元组 $(A,B)$，输出 -1，否则输出一行两个整数 $A,B$。

$|X|,|Y|\le 10^{17},(X,Y)\not= (0,0)$

输入格式

入力は以下の形式で標準入力から与えられる。

$X\ \ Y$

输出格式

条件を満たす整数の組 $(A,\ B)$ が存在する場合は以下の形式で出力せよ。

$A\ \ B$

条件を満たす整数の組 $(A,\ B)$ が存在しない場合は -1 を出力せよ。

制約

  • $-10^{17}\le X,Y\le 10^{17}$
  • $(X,\ Y)\ \ne\ (0,\ 0)$
  • $X,\ Y$ は整数

Solution

我们先使用行列式计算三角形的面积。

$$
\frac{1}{2}\begin{vmatrix}0&0&1\X&Y&1\A&B&1\end{vmatrix}=1
$$

我们将其化简得到：$BX-AY=\pm 2$。观察到这很类似于扩展欧几里德 $ax+by=\gcd(a,b)$。

因为 $A,B$ 都是正整数，如果 $\gcd(X,Y)\gt 2$，那么就是无解的。否则，我们可以先找到一组 $(A_1,B_1)$ 满足 $BX-AY=\gcd(X,Y)$，然后再两边同乘 $\frac{2}{\gcd(X,Y)}$ 就可以得到一组合法的答案了。

AC Code

# 扩展欧几里德 (exgcd)
