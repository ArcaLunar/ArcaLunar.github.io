CF1219C - 周期整数

Problem Description

Alice became interested in periods of integer numbers. We say positive XXX integer number is periodic with length LLL if there exists positive integer number PPP with LLL digits such that XXX can be written as PPPP…PPPPP…PPPPP…P. For example:

X=123123123X = 123123123X=123123123 is periodic number with length L=3L = 3L=3 and L=9L = 9L=9

X=42424242X = 42424242X=42424242 is periodic number with length L=2,L=4L = 2,L = 4L=2,L=4 and L=8L = 8L=8

X=12345X = 12345X=12345 is periodic number with length L=5L = 5L=5

For given positive period length LLL and positive integer number AAA, Alice wants to find smallest integer number XXX strictly greater than AAA that is periodic with length LLL.

Input

First line contains one positive integer number L (1≤L≤105)L \ (1 \leq L \leq 10^5)L (1≤L≤105) representing length of the period. Second line contains one positive integer number A (1≤A≤10100000)A \ (1 \leq A \leq 10^{100 000})A (1≤A≤10100000).

Output

One positive integer number representing smallest positive number that is periodic with length LLL and is greater than AAA.

Solution

  3. 否则 N%L=0N\%L=0N%L=0，因为要生成的數最小，贪心地考虑，尝试先取前 LLL 个数字组成 XXX，并将 XXX 重复。此时仍有三种情况：
    1. 如果不是 X=99…9X=99\ldots9X=99…9 的形式
      1. 先看 XXX 重复多次的情况
      2. 那么可以令 X′=X+1X' = X+1X′=X+1，这样新数在第 LLL 位大于 AAA，且尽可能的小。
