Lecture 2. Linear Regression, Batch/Stochastic Gradient Descent

用概率论解释为什么 cost function 是平方？

我们假设的预测函数 y(i)=θTx(i)+ϵ(i)y^{(i)}=\theta^Tx^{(i)}+\epsilon^{(i)}y(i)=θTx(i)+ϵ(i)，其中 ϵ(i)\epsilon^{(i)}ϵ(i) 是没有算入模型的误差项。我们可以假设 ϵ(i)\epsilon^{(i)}ϵ(i) 服从正态分布 ϵ(i)∼N(0,σ2)\epsilon^{(i)}\sim\mathcal N(0,\sigma^2)ϵ(i)∼N(0,σ2)，并且是互相独立地同等分布（Independently and Identically Distributed, IID），因此其概率分布为

p(ϵ(i))=12πσexp⁡(−(ϵ(i))22σ2)p(\epsilon^{(i)})=\frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(\epsilon^{(i)})^2}{2\sigma^2}) p(ϵ(i))=2π​σ1​exp(−2σ2(ϵ(i))2​)

这意味着

p(y(i)∣x(i);θ)=12πσexp⁡(−(y(i)−θTx(i))22σ2)p(y^{(i)}|x^{(i)};\theta)=\frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(y^{(i)}-\theta^Tx^{(i)})^2}{2\sigma^2}) p(y(i)∣x(i);θ)=2π​σ1​exp(−2σ2(y(i)−θTx(i))2​)

这个公式的意思是，当我的样本 x(i)x^{(i)}x(i) 确定时，由于误差 ϵ(i)\epsilon^{(i)}ϵ(i) 是不确定的，所以 y(i)y^{(i)}y(i) 也是不确定的，并且他们之间只差一个常数 θTx(i)\theta^Tx^{(i)}θTx(i)，因此也可以说 (y(i)∣x(i);θ)∼N(θTx(i),σ2)(y^{(i)}|x^{(i)};\theta)\sim\mathcal N(\theta^Tx^{(i)},\sigma^2)(y(i)∣x(i);θ)∼N(θTx(i),σ2)。

给定样本矩阵 XXX 和参数 θ\thetaθ，y(i)y^{(i)}y(i) 的分布可以看作关于 y⃗\vec yy​ 的函数 p(y⃗∣X;θ)p(\vec y|X;\theta)p(y​∣X;θ)。但我们更像把它看成关于 θ\thetaθ 的函数，所以我们定义 Likelihood function

\begin{align*} L(\theta)=L(\theta|X,\vec y)&=p(\vec y|X;\theta)\\ &=\prod_{i=1}^n p(y^{(i)}|x^{(i)};\theta)\tag{Expand \(X,Y\) into Vectors}\\ &=\prod_{i=1}^n \frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(y^{(i)}-\theta^Tx^{(i)})^2}{2\sigma^2}) \end{align*}

那么我们如何选取最好的 θ\thetaθ 呢？Maximum Likelihood Principal 告诉我们，我们应该选择 θ\thetaθ 使得 L(θ)L(\theta)L(θ) 最大。

为了简化计算，我们定义 Log Likelihood Function l(θ){l}(\theta)l(θ)

\begin{align*} l(\theta)&=\ln L(\theta)\tag{Definition}\\ &=\ln \prod_{i=1}^n \frac{1}{\sqrt{2\pi}\sigma}\exp(-\frac{(y^{(i)}-\theta^Tx^{(i)})^2}{2\sigma^2})\\ &=n\ln{\frac{1}{\sqrt{2\pi}\sigma}}-\frac{1}{\sigma^2}\cdot\frac12\sum_{i=1}^n(y^{(i)}-\theta^Tx^{(i)})^2 \end{align*}

而后面这一部分就是我们的 cost function J(θ)J(\theta)J(θ)。

Lecture 3. Locally Weighted & Logistic Regression, Feature Selection Algorithm

Categories of Learning Algorithms

  2. Non-Parametric Learning Algorithm：参数数量会变（通常随数据集增大而增多）

Locally Weighted Regression

对于要预测的 xxx，我们找到样本中离 xxx 最近的若干个 sample，然后利用这些 sample 在局部进行 Linear Regression，用局部拟合出来的直线进行预测。

用数学语言描述，我们仍然是求参数 θ\thetaθ 使得修改过的 cost function J(θ)J(\theta)J(θ) 最小

J(θ)=∑i=1MW(i)(θTX(i)−Y(i))2J(\theta)=\sum_{i=1}^M W^{(i)}\Big(\theta^TX^{(i)}-Y^{(i)}\Big)^2 J(θ)=i=1∑M​W(i)(θTX(i)−Y(i))2

其中的 W(i)W^{(i)}W(i) （也是矩阵）被称为 weight function，通常我们取

W(i)=exp⁡(−(X(i)−X)2τ2)W^{(i)}=\exp\Big(-\frac{(X^{(i)}-X)^2}{\tau^2}\Big) W(i)=exp(−τ2(X(i)−X)2​)

超参数 τ\tauτ （tau）决定了“局部”的大小。

Logistic Regression

Logistic Regression 适合分类任务。这里我们先只讨论二分类 Binary Classification，即 h:Rn↦{0,1}h:\R^n\mapsto \{0,1\}h:Rn↦{0,1}，当 y(i)=1y^{(i)}=1y(i)=1 时称为 positive class，为 000 时称为 negative class。

如果我们沿用 Linear Regression 的方法来预测 Logistic Regression，很容易构造数据集使得这种方法给出错误的预测结果。所以，我们需要换一个 hypothesis hθ(x)h_\theta(x)hθ​(x)

hθ(x)=g(θTx)=11+e−θTxh_\theta(x)=g(\theta^Tx)=\frac{1}{1+e^{-\theta^Tx}} hθ​(x)=g(θTx)=1+e−θTx1​

其中 g(z)=11+exp⁡(z)g(z)=\frac{1}{1+\exp(z)}g(z)=1+exp(z)1​ 被称为 Logistic Function 或者 Sigmoid Function，而且有 g′(z)=g(z)(1−g(z))g'(z)=g(z)(1-g(z))g′(z)=g(z)(1−g(z))

我们先作出一系列假设，然后按上一节的概率论方法 Maximum Likelihood 来求 θ\thetaθ。

我们定义

\begin{align*} p(y=1|x;\theta)&=h_\theta(x)\\ p(y=0|x;\theta)&=1-h_\theta(x) \end{align*}

可以统一起来

p(y∣x;θ)=(hθ(x))y(1−hθ(x))1−yp(y|x;\theta)=(h_\theta(x))^y(1-h_\theta(x))^{1-y} p(y∣x;θ)=(hθ​(x))y(1−hθ​(x))1−y

由定义，Likelihood Function 为

\begin{align*} L(\theta)&=\prod_{i=1}^n \big(h_\theta(x^{(i)})\big)^{y^{(i)}}\big(1-h_\theta(x^{(i)})\big)^{1-y^{(i)}} \end{align*}

Log Likelihood Function 为

\begin{align*} l(\theta)&=\sum_{i=1}^n \Big(y^{(i)}\ln h(x^{(i)})+(1-y^{(i)})\ln\big(1-h(x^{(i)})\big)\Big) \end{align*}

我们这里也可以用 Gradient Ascent 类似的思路，我们用 θt+1←θt+α⋅∇θl(θ)\theta_{t+1}\gets \theta_t+\alpha\cdot\nabla_\theta l(\theta)θt+1​←θt​+α⋅∇θ​l(θ) 来迭代，而

\begin{align*} \nabla_{\theta_j}l(\theta)&=\nabla_{\theta_j}(y\ln h(x)+(1-y)\ln(1-h(x)))\\ &=\nabla_{\theta_j}(y\ln g(\theta^Tx)+(1-y)\ln(1-g(\theta^Tx)))\tag*{\(h_\theta(x)=g(\theta^Tx)\)}\\ &=(\frac{y}{g(\theta^Tx)}+\frac{1-y}{1-g(\theta^Tx)})g(\theta^Tx)(1-g(\theta^Tx))x_j ,\tag*{\(g'(z)=g(z)(1-g(z))\)}\\ &=(y-h_\theta(x))x_j \end{align*}

θt+1:=θt+α⋅(y(i)−hθ(x(i)))xj(i)\theta_{t+1}:= \theta_t+\alpha\cdot(y^{(i)}-h_\theta(x^{(i)}))x_j^{(i)} θt+1​:=θt​+α⋅(y(i)−hθ​(x(i)))xj(i)​

也被称作 Perceptron Learning Algorithm

牛顿迭代法 a←a−f(x)f′(x)a\gets a-\frac {f(x)}{f'(x)}a←a−f′(x)f(x)​ 可以快速找到零点，所以令 f(θ)=l′(θ)f(\theta)=l'(\theta)f(θ)=l′(θ)，我们可以利用来快速迭代
